# Support Bot — Telegram-бот поддержки пользователей

Бот для службы поддержки: пользователи пишут вопросы в Telegram, администратор получает уведомления и отвечает прямо в боте.

Автор: **maxim_vdonsk**

---

## Что умеет бот

- Принимает текстовые сообщения и фотографии от пользователей
- Уведомляет администратора с кнопкой «Открыть диалог»
- Администратор отвечает пользователю, не выходя из Telegram
- История диалога сохраняется в базе данных SQLite
- Диалог можно пометить как «Решено» — пользователь получит уведомление

## Быстрый старт

### 1. Получите токен бота

1. Напишите боту [@BotFather](https://t.me/BotFather) в Telegram
2. Выберите `/newbot` и следуйте инструкциям
3. Скопируйте токен вида `123456789:AAF...`

### 2. Узнайте свой Telegram ID

Напишите боту [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш числовой ID.

### 3. Клонируйте репозиторий

```bash
git clone https://github.com/maxim_vdonsk/support_bot.git
cd support_bot
```

### 4. Создайте виртуальное окружение

```bash
# Создать окружение
python -m venv venv

# Активировать (macOS / Linux)
source venv/bin/activate

# Активировать (Windows)
venv\Scripts\activate
```

> Виртуальное окружение изолирует зависимости проекта от системного Python.
> Команда `(venv)` в начале строки терминала означает, что окружение активно.

### 5. Установите зависимости

```bash
pip install -r requirements.txt
```

### 6. Создайте файл с секретами

```bash
cp .env.example .env
```

Откройте `.env` и заполните значения:

```env
BOT_TOKEN=123456789:AAF_ваш_токен_здесь
ADMIN_ID=123456789
```

> Файл `.env` добавлен в `.gitignore` — он не попадёт в git, токен останется только у вас.

### 7. Запустите бота

```bash
python bot.py
```

В терминале появится сообщение `Бот запущен`. Найдите вашего бота в Telegram и напишите `/start`.

## Команды бота

| Команда | Кто использует | Описание |
|---|---|---|
| `/start` | Пользователь | Приветствие и инструкция |
| `/dialog <user_id>` | Администратор | Открыть диалог с пользователем по ID |

### Как отвечать пользователям

1. Когда пользователь напишет боту — вы получите уведомление с кнопкой **«Открыть диалог»**
2. Нажмите кнопку — бот покажет историю переписки
3. Просто напишите следующее сообщение боту — оно уйдёт выбранному пользователю
4. Когда вопрос решён — нажмите кнопку **«Решено»**

## База данных

При первом запуске автоматически создаётся файл `support.db` со следующей структурой:

| Поле | Тип | Описание |
|---|---|---|
| `id` | INTEGER | Автоматический номер записи |
| `user_id` | INTEGER | Telegram ID пользователя |
| `username` | TEXT | @username |
| `fullname` | TEXT | Полное имя |
| `role` | TEXT | `user` или `admin` |
| `message` | TEXT | Текст сообщения |
| `photo` | TEXT | file_id фотографии (если есть) |
| `status` | TEXT | `ожидание` или `решено` |
| `time` | TIMESTAMP | Время сообщения |

## Возможные проблемы

**Бот не запускается — ошибка `BOT_TOKEN не задан`**
→ Проверьте, что файл `.env` существует и в нём правильно заполнен `BOT_TOKEN`.

**Сообщение пользователю не доходит**
→ Убедитесь, что `ADMIN_ID` в `.env` — это именно числовой ID, а не @username.

**`ModuleNotFoundError: No module named 'telegram'`**
→ Убедитесь, что активировано виртуальное окружение и выполнен `pip install -r requirements.txt`.
